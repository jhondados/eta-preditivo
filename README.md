## 📊 Resultados-chave (KPIs)
**96.8%** Acurácia · **2.4M/dia** Entregas · **-28%** Atraso · **+19%** CSAT

## 🧠 Tecnologias de IA
ML · LightGBM · Real-time

## ✨ O que faz
- ETA em tempo real
- Fatores: trânsito, clima
- Alertas de atraso
- Reotimização de prazo

## 🗄️ Fonte de dados real
[ANTT + clima + trânsito](https://www.gov.br/antt)

## ⚙️ Como funciona
1. **Coleta de dados reais** — integração com ANTT + clima + trânsito
2. **Processamento com IA** — ML · LightGBM · Real-time
3. **Dashboard & insights** — KPIs, gráficos e recomendações acionáveis
4. **Ação automatizada** — alertas, predições e decisões em tempo real
